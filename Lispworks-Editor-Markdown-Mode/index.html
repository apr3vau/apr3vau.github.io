Lispworks Editor Markdown Mode

Lispworks Editor Markdown Mode

Provide markdown syntax highlight, maybe buggy. Wrote for improving my writing experience.

Source Code

Usage

The code needs CL-PPCRE to work, so please load CL-PPCRE or QUICKLISP first, then save the source code, edit the Easy Custom Settings part to suit your own environment, finally compile & load the file.

Editor command Markdown Mode will be provided, and Font Lock Fontify ... will work for markdown file.
